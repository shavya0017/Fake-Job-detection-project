# 🕵️ Fake Job Detection using Machine Learning

## 📌 Project Overview
This project uses Machine Learning to classify job postings as **Real** or **Fake**. It helps identify fraudulent job advertisements by analyzing the job description and related information.

## 🚀 Features
- Detects fake and genuine job postings
- Data preprocessing and cleaning
- Feature engineering
- Machine Learning model training
- Prediction on custom job descriptions

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn

## 📂 Dataset
- Fake Job Postings Dataset

## ▶️ How to Run
1. Clone this repository.
2. Install the required libraries.
3. Open `Fake_Job_Detection.ipynb`.
4. Run all cells in Jupyter Notebook.

## 📊 Output
The model predicts whether a job posting is:
- ✅ Legitimate Job Posting
- ❌ Fake Job Posting
